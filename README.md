# 習慣養成工廠 Habit Tycoon

仿《美少女夢工廠》養成系統，把柯維「七個習慣」改編成國中八年級學生的養成遊戲。
玩家用一年 52 週，把一個八年級孩子養成具備成功品格的人，結局由規則引擎依「精通了哪些習慣」判定。

## 玩法

- **角色設定**：自訂名字、選男生／女生。
- **課程系統**：7 個習慣 × 5 個知識點 = 35 個知識點。
  - 自律群（個人成功）：習慣 1 主動積極、2 以終為始、3 要事第一
  - 人際群（公眾成功）：習慣 4 雙贏思維、5 知彼解己、6 統合綜效
  - 持續更新群（韌性）：習慣 7 不斷更新
- **每週流程**：選一個知識點 → 做 10 題情境選擇題 → 答對 ≥ 8 題即「精通」該知識點（習慣值 +10）；未達門檻則該週空轉，下週可再挑戰。
- **題庫**：350 題（每習慣 50 題），全為八年級真實生活情境，每題 4 選項、1 正解、附回饋。
- **結局**：第 52 週結束或 35 點全精通時觸發，**規則引擎（非 LLM）**判定 30 種結局之一：
  - 成功 20 種：A 均衡圓滿 / B 自律精進 / C 人際共好 / D 韌性更新（各 5 級）
  - 失敗／偏差 10 種：F1–F10，對應缺某習慣或失衡（如「孤獨的資優生」＝自律強但缺習慣五）
  - 結局報告含：七習慣雷達圖、性格側寫、職業推測、人生陷阱警告、給孩子的真心話。

## 技術

- Next.js 16（App Router）+ TypeScript + Tailwind CSS 4
- 純前端、client-only：進度存 `localStorage`，無後端（遊客版）
- 結局判定為純函式規則引擎（`lib/endings.ts`），可單元測試、無 API 成本

## 開發

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # production build
```

## 結構

```
data/            350 題題庫（habit1–7.json）+ 30 結局（endings.json）
lib/
  types.ts       共用型別
  habits.ts      7 習慣 × 35 知識點定義
  questions.ts   題庫載入 / 依知識點取題
  endings.ts     30 結局資料 + judgeEnding 規則引擎
  game.ts        常數、localStorage 存取
app/page.tsx     遊戲主畫面（標題 / 設定 / 週 / 測驗 / 結果 / 結局）
```

## 可調參數

- `lib/game.ts`：`TOTAL_WEEKS`（52）、`QUIZ_SIZE`（10）、`PASS_THRESHOLD`（8，過關門檻）
- `lib/endings.ts`：`judgeEnding` 內的成功門檻與失敗判定邏輯
